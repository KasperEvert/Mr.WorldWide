Is it a difficult script to learn?

There are some diffrences between for example english & european alphabets and the Arabic alphabet that will make it hard to learn. For example, vowles aren't used in the same way in Arabic as in English. However, if you already speak Arabic or if you already know some other semitic script it shouldn't be too hard.

What does الْأَبْجَدِيَّة - al-abjadīyah mean?

It comes from the order of the letters in the alphabet. Just like how the English word "alphabet" comes from the first two letters of the Greek alphabet, Alpha and Beta, the Arabic Abjad comes from the first four letters of the alphabet. Specifically the letters: ʾalif, bāʾ, jīm & dāl which depending on order comes first. When read as a word the entire alphabet reads as such:
ʾabjad hawwaz ḥuṭṭī kalaman saʿfaṣ qarashat thakhadh ḍaẓagh

أَرْقَام هِنْدِيَّة - Oriental Arabic–Hindu Numerals

Note: Arab numerals are written from left to right, but if you have a collection of oriental arab numerals you start from the right and go left (just like you read arabic text), but each numeral cluster is read from left to right (just like western arabic numerals). An example is given with the number "10" on each line.

Western01234 5678910
Arab ٠١٢ ٣٤٥ ٦٧٨ ٩١٠
Persian ۰۱۲ ۳۴۵ ۶۷٨ ٩۱۰

Example1: ٱلشَّهَادَةُ - Aš-Šahādah

The Shahadah - The Islamic oath - is an important phrase according to probably most Arabic speakers, and it looks like this:

.أَشْهَدُ أَنْ لَا إِلَٰهَ إِلَّا ٱللَّٰهُ وَأَشْهَدُ أَنَّ مُحَمَّدًا رَسُولُ ٱللَّٰهِ

It can be transliterated as such: ašhadu ʾalā ʾilāha ʾilla -llāhu, wa-ʾašhadu ʾanna muḥammadan rasūlu -llāh. And can roughly be translated as: I bear witness that there is no deity but God, and I bear witness that Muhammad is his messenger. This phrase can be found on flags like the Saudi Arabian flag and the Taliban Afghanistan flag.

Nastaʿlīq: أَشْهَدُ أَنْ لَا إِلَٰهَ إِلَّا ٱللَّٰهُ وَأَشْهَدُ أَنَّ مُحَمَّدًا رَسُولُ ٱللَّٰهِ

Thuluth: أَشْهَدُ أَنْ لَا إِلَٰهَ إِلَّا ٱللَّٰهُ وَأَشْهَدُ أَنَّ مُحَمَّدًا رَسُولُ ٱللَّٰهِ


Example2: تَكْبِير - The Takbir

The Takbir is a phrase that is used by Arabic speaking monotheist, for the most part Christians and Muslims. Takbir means magnification [of God]. The phrase can for example be found on the flag of Iraq. The phrase is as such:

ٱللَّٰهُ أَكْبَرُ

The phrase transliterats as ʾAllāhu ʾakbar, and means "[The] God is the greatest".

Nastaʿlīq: ٱللَّٰهُ أَكْبَرُ

Thuluth: ٱللَّٰهُ أَكْبَرُ
