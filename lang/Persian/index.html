Is it a difficult script to learn?

There are some diffrences between for example english & european alphabets and the Persian alphabet that will make it hard to learn. For example, vowles aren't used in the same way as in English. However, if you already speak Persion or if you already know some other semitic script it shouldn't be too hard.

What does الفبای - "alefbā-ye" mean?

It comes from the first two letters of the alphabet, just like how the English word "Alphabet" comes from the first two letters of the Greek alphabet: Alpha, beta.

شماره - Oriental Arabic–Hindu Numerals

Note: Arab numerals are written from left to right, but if you have a collection of oriental arab numerals you start from the right and go left (just like you read arabic text), but each numeral cluster is read from left to right (just like western arabic numerals). An example is given with the number "10" on each line.

Western01234 5678910
Persian ۰۱۲ ۳۴۵ ۶۷٨ ٩۱۰
Arab ٠١٢ ٣٤٥ ٦٧٨ ٩١٠
